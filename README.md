# TaskFlow | Focus & Create

TaskFlow is a premium, glassmorphism-inspired Todo application designed to help you stay focused and productive. It features a sleek, interactive interface with real-time countdowns, progress tracking, and robust task management.

##  Features

###  Core Functionality
- **Dynamic Task Creation**: Add tasks with descriptions, due dates, due times, and priority levels.
- **Glassmorphism UI**: Beautifully designed interface with vibrant colors, subtle gradients, and smooth animations.
- **Real-time Countdowns**: Each task shows a live countdown until its deadline.
- **Progress Tracking**: A visual progress bar at the top tracks your completion rate across all tasks.
- **Persistent Storage**: All tasks are saved locally in your browser, so you never lose your progress.
 
###  Advanced Task Management (New Additions)
- **Advanced Editing**: Modify any part of an existing task—description, deadline, or priority—via a dedicated edit modal.
- **Priority Badging**: Color-coded badges (High, Medium, Low) to help you focus on what matters most.
- **Description Toggling**: Long task descriptions are automatically truncated with a "Show More/Less" toggle to keep the list clean.
- **Clear Completed**: One-click cleanup of finished tasks.

### ⌨ Keyboard Accessibility (Power Users)
Manage your workflow entirely from your keyboard:
- **`Tab`**: Navigate between tasks.
- **`Enter` / `Space`**: Toggle task completion.
- **`Delete` / `Backspace`**: Remove a task.
- **`E`**: Edit the focused task.
- **`Escape`**: Close the edit modal without saving.
- **`Enter` (in Modal)**: Save changes from any input field.

##  Technology Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS (Custom properties, Flexbox, CSS Grid)
- **Logic**: ES6+ JavaScript (Modules)
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Build Tool**: Vite

##  Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

##  Responsive Design
TaskFlow is fully responsive and optimized for all screen sizes, from desktop monitors to mobile phones.
